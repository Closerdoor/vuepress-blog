## create-react-app 环境搭建
```
npx create-react-app project-name
npm i redux -S
npm i react-redux -S //把react-redux进行整合
npm i redux-thunk -S //异步更新redux里的数据
npm i react-router-dom -S //路由
npm i axios -S 
npm i antd -S // UI库
```
